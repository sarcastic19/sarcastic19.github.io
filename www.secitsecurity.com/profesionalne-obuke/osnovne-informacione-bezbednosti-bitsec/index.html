Osnovna obuka informacione bezbednosti “BITSEC”

Instruktor

Instruktor ove obuke je prof. dr Igor Franc.

Kome je namenjena obuka

Obuka je primarno namenjena sistem administratorima, administratorima mreže i Database administratorima.

Trajanje

Obuka traje 2 radna dana po 4 školska časa, sa početkom u dogovoreno vreme.

Potreban nivo predznanja

Za ovu obuku je potreban osnovni nivo predznanja u oblasti informacionih tehnologija.

Prijava

Prijavu za obuku možete pronaći na ovom linku.

Sadržaj

Teme koje će da se obrade na obuci su:

  1. Training Introduction
  2. Security and Functionality (CIA)
  3. Authentication and Authorization – Basic Concepts
  4. Types of Attacks and Threats
  5. Malware (Virus, Worm, Trojan Horse…)
  6. Social Engineering
  7. Defence from Diferent Attack Types
  8. Windows Security
  9. Linux Security
  10. Network Security
  11. Wireless Security
  12. Mobile Devices Security (Smart Phone, Tablet, Laptop)
  13. Mail Security
  14. Web Security
  15. Database Security
  16. Application and Software Security
  17. Backup and Data Protection (Storage and Deleting of Data)
  18. Users and Access Rights (Access Control, Passwords, Accounts…)
  19. Cryptography Basics (Crypt, Algorithm, PKI…)
  20. Physical Security (Building, Stuff, Equipment…)
  21. Rules and Procedures in Data Security
  22. Human Factors (Training, Mandatory Holidays, Job Rotation, Separation of Duties)
  23. The Legal and Ethical Standards in Security
